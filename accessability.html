Accessability

Reflections on accessibility

An important principle when designing for accessibility is that the page should work for everyone, without need for adaptation. This is something we wanted to implement when designing, and led to us making certain decisions: one of them was the choice to use a sans-serif font, to make it easier to read for everyone. We also made sure that there were sufficient contrasts, to improve readability. One last thing we did while coding, was to improve captions for all pictures, to make the use of a screen reader easier. As this is a small project, we did not have time to test our website on many different users, which is something that might have improved the accessibility.
